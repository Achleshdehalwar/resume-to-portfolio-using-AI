Portfolio Website Generator

A Streamlit-based tool that converts a resume (PDF or DOCX) into a clean, responsive personal portfolio website using HTML, CSS, and JavaScript. The app extracts structured resume content and generates a multi-section static website ready for download.

Features

Upload PDF or DOCX resumes

Extract and structure content into sections: Header, About, Skills, Experience, Projects, Education, Contact

Outputs standalone HTML, CSS, and JavaScript

Download website files as a ZIP

Simple, developer-focused UI

Tech Stack

Python, Streamlit, LangChain, Google Gemini API, PyPDF2, python-docx
